Замечание!

Этот проект применяет мощь Markdown, преобразуя обычный текст в структурированный HTML. В следующих разделах вы узнаете, что такое Markdown и как эффективно использовать его в рамках этого проекта. Рассматривайте эту страницу как ваше персональное руководство по навигации по проекту.

Что такое Markdown?

Markdown – это легкий язык разметки, созданный с целью облегчения добавления форматирования к тексту в интернете. Он позволяет вам преобразовывать текст в HTML или XHTML и многие другие форматы.

Основная идея Markdown – сделать текст читаемым и легко редактируемым, независимо от того, преобразован он в другой формат или нет. Это достигается за счет использования простого и интуитивно понятного синтаксиса.

Markdown широко используется для создания документации, написания сообщений на форумах, создания контента для сайтов и многое другое. Он поддерживается многими платформами публикации контента, такими как GitHub, GitLab, Bitbucket, Jekyll, и многими другими.

Markdown поддерживает различные элементы форматирования, такие как заголовки, ссылки, списки, выделение текста, и многое другое.

Примеры использования Markdown

Заголовки

В Markdown заголовки создаются с помощью символа решетки #. Количество решеток перед заголовком определяет его уровень. Вот примеры:

# Заголовок 1
## Заголовок 2
### Заголовок 3
#### Заголовок 4
##### Заголовок 5
###### Заголовок 6

Параграфы

В Markdown параграфы создаются с помощью одного или более последовательных строк текста, разделенных одной или более пустыми строками. Вот пример:

Это первый параграф.

Это второй параграф.

Если вы хотите создать перенос строки без начала нового параграфа, вы можете использовать два или более пробела в конце строки:

Это строка с переносом.
Это следующая строка.

Экранирование

Экранирование символов в Markdown — это способ указать, что определенные символы следует интерпретировать буквально, а не как часть синтаксиса Markdown. Это полезно, когда вам нужно отобразить символы, которые обычно используются для форматирования текста, например * или _, без применения их форматирующего эффекта.

Чтобы экранировать символ, вы ставите перед ним обратную косую черту (\). Вот примеры:

\\   обратная косая черта
\`   обратный апостроф
\*   звездочка
\_   нижнее подчеркивание
\{   фигурная скобка открывающая
\}   фигурная скобка закрывающая
\[   квадратная скобка открывающая
\]   квадратная скобка закрывающая
\(   круглая скобка открывающая
\)   круглая скобка закрывающая
\#   решетка
\+   плюс
\-   минус (или дефис)
\.   точка
\!   восклицательный знак

Выделения текста

В Markdown отдельные части текста можно выделять разные способами. Например, жирным, курсивом, моноширинным, зачеркнутым или подчеркнутым и т.д.

Для выделения курсивом используются, либо одна звездочка "*", либо одно подчеркивание "_". Вот как это выглядит:

*Курсив*
_Курсив_

Для выделения жирным используются, либо две звездочки "**", либо два подчеркивания "__". Вот как это выглядит:

**Жирный текст**
__Жирный текст__

Для выделения жирным курсивом используются, либо три звездочки "***", либо три подчеркивания "__". Вот как это выглядит:

***Жирный курсив***
___Жирный курсив___

Для выделения моноширинным текстом используются обратные апострофы "`". Вот как это выглядит:

`Моноширинный текст`

Для выделения зачеркнутого текста не существует специального символа. Однако, можно воспользоваться HTML тэгом <del>. Вот как это выглядит:

<del>Привет мир!<del>

Для выделения подчеркнутого текста не существует специального символа. Однако, можно воспользоваться HTML тэгом <u>. Вот как это выглядит:

<u>100 рублей!<u>

Цитаты

В Markdown цитаты создаются с помощью символа > перед текстом. Имеется возможность использовать вложенные цитаты, комбинируя символы >. Вот пример:

> # Заголовок
> Первый параграф
>
> Второй параграф
>
> > Вложенная цитата
> > > Цитата третьего уровня
>
> Продолжение основной цитаты

Нумерованные списки

В Markdown нумерованные списки создаются с помощью чисел, за которыми следует точка. Вот пример:

1. Один
2. Два
3. Три

Маркированные списки

В Markdown маркированные списки создаются с помощью символа звездочки "*". Вот пример:

* Первый пункт
* Второй пункт
* Третий пункт

Ссылки

Ссылки в Markdown имеют несколько видов: встроенные, с всплывающей подсказкой, автоматические ссылки url.

Встроенные ссылки состоят из [...] - текст ссылки, и (...) - url ссылки. Вот пример:

[Google](https://google.com/)

Ссылки с всплывающей подсказкой похожи на встроенные ссылки. Чтобы указать текст подсказки, необходимо написать его в кавычках после url ссылки. Вот пример:

[Google](https://google.com/ "Ссылка на Google")

Ну и наконец автоматические ссылки url, которые выглядят как обычный url адрес. Достаточно окружить ссылку треугольными скобками <>. Вот пример:

<https://google.com/>

Код

В Markdown код помещается между тройными обратными кавычками ```. После открывающих кавычек можно указать язык программирования для активации подсветки синтаксиса. И, конечно, не забудьте добавить закрывающие кавычки ```. Вот как это выглядит на практике:

```python
def hello_world():
    print("Hello, world!")
```
